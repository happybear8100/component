Components
Component Description
mdl-expansion-panel-group Container for grouping multiple panels.
mdl-expansion-panel Defines a panel. It can exist on it's own or can be inside a group.
mdl-expansion-panel-header Defines the header that is shown at the top of each panel. It has a toggle button which expands the panel.
mdl-expansion-panel-header-list-content Defines title for the panel's header.
mdl-expansion-panel-header-secondary-content Defines secondary title for the panel's header.
mdl-expansion-panel-content Container for the expandable content inside the panel.
mdl-expansion-panel-body Defines the collapsible content for the panel
mdl-expansion-panel-content-footer Defines any action buttons for the panel.
API for MdlExpansionPanelComponent
Function Params Returns
toggle none void
expand none void
collapse none void
disableToggle none void
enableToggle none void
Inputs for MdlExpansionPanelComponent
Property Params Explanation
disabled boolean Panel initializes in disabled state
expanded boolean Panel initializes in expanded state
API for MdlExpansionPanelGroupComponent
Function Params Returns
getExpanded none

A number representing the index of expanded panel.

-1 if none of the panel is expanded.
getPanel number - index of the panel in the group (0-based) an instance of MdlExpansionPanel at the specified index
