Component Description
mdl-fab-menu Container hosting the toggle button and the popover.
mdl-fab-menu-item Defines a single item of the fab-menu. It takes a reference to the enclosing menu, the name of the icon to be displayed on the mini-fab and a label for the tooltip.
Inputs for MdlFabMenuComponent
Property Params Explanation
alwaysShowTooltips boolean Set if either or not the tooltip should always be displayed when the menu is up
Inputs for MdlFabMenuItemComponent
Property Params Explanation
icon string name of the icon to be used in the menu item
label string text of the tooltip
Output for MdlFabMenuItemComponent
Property Params Explanation
menu-clicked event<void> event triggered when this item has been selected
